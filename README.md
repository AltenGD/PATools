# PATools
Some tools for Project Arrhythmia

## Streamer Tools
This is a tool to help streamers have a dynamic ~~not really~~ stream with changing beatmap

### File Structure
it is ESSENTIAL to have this exact structure to make sure everything is working correctly
```
PAStream:
    Beatmap:
            artist:
                    link.txt
                    name.txt
            beatmap:
                    date_edited.txt
                    game_version.txt
                    version_number.txt
                    workshop_id.txt
            creator:
                    steam_id.txt
                    steam_name.txt
            song:
                    bpm.txt
                    description.txt
                    difficulty.txt
                    t.txt
                    title.txt
```
